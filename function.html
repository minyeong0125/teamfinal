프론트엔드란?

프론트엔드 핵심 기술 소개

HTML

웹사이트의 구조와 내용(뼈대)을 만듭니다. 태그(<h1>, <p>, <a> 등)를 사용해 제목, 문단, 링크, 이미지 등을 표시합니다.

CSS

웹사이트의 스타일과 레이아웃(옷, 디자인)을 꾸밉니다. 색상, 폰트, 크기, 간격, 배치 등 시각적 요소를 제어합니다.

JavaScript

웹사이트의 동작과 상호작용(움직임, 기능)을 구현합니다. 버튼 클릭, 폼 제출, 애니메이션, 데이터 처리 등 다양한 동적 기능을 제공합니다.
