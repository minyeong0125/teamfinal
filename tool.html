Tool Information

Visual Studio Code

소스코드 편집기
웹 개발 시 가장 많이 사용되는 소스코드 편집기이다. HTML, CSS, JavaScript뿐만 아니라 PHP, Node.js 등 다양한 언어를 지원하며, 실시간 디버깅과 Git연동 등 다양한 기능들이 내장되어있다. 효율적인 개발 환경을 제공받을 수 있는 소스코드 편집기이다.

Bootstrap

웹 사이트를 쉽게 만들 수 있게 도와주는 CSS, JS 프레임워크
트위터에서 개발한 CSS 프레임워크로, 반응형 웹 디자인을 손쉽게 구현할 수 있게 도와준다. 미리 정의된 스타일과 컴포넌트를 활용해 복잡한 레이아웃도 간편하게 만들 수 있으며, 웹사이트의 뼈대를 빠르게 완성할 수 있는 강력한 도구이다.

GitHub

저장소 호스팅 서비스 및 공동 협업 플랫폼
소스코드를 저장하고 버전을 관리할 수 있는 플랫폼으로, 전 세계 개발자들이 협업하고 소통할 수 있는 공간이다. 코드의 변경 이력을 관리할 수 있고 프로젝트를 다른 사람과 공유하거나 팀원들과 함께 개발할 수 있어 협업 필수 도구로 자리잡았다.

Vercel

클라우드 컴퓨팅 서비스
정적 웹사이트 및 프론트엔드 프로젝트 배포를 위한 플랫폼으로, 복잡한 서버 설정 없이도 몇 번의 클릭만으로 프로젝트를 세상에 공개할 수 있게 해준다. GitHub와 연동하면 푸시할 때마다 자동 배포가 되어 편리하다.
